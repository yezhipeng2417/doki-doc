# Platform Learning 讲义总目录

## 目录结构说明
1. 所有讲义都在同一个文件夹下。
2. 文件名前缀 `01-12` 代表建议学习顺序。
3. 每份讲义按知识点模块组织，不按 Day 组织。

## 讲义文件
1. `01-aws-docker-k8s-foundation.md`：AWS + Docker + Kubernetes 基础
2. `02-container-k8s-advanced.md`：容器进阶 + Kubernetes 实操基础
3. `03-eks-intro-practice.md`：EKS 入门实战
4. `04-eks-networking-security.md`：EKS Networking + Security
5. `05-cicd-github-actions.md`：CI/CD + GitHub Actions
6. `06-gitops-helm-rollback.md`：GitOps + Helm + 回滚
7. `07-observability.md`：可观测性
8. `08-security-cost-governance.md`：安全与成本治理
9. `09-bedrock-api-foundation.md`：Bedrock 基础与 API 实战
10. `10-guardrails-security-policies.md`：Guardrails 与安全策略
11. `11-agents-agentcore.md`：Agents / AgentCore
12. `12-final-project-defense.md`：最终项目与答辩

## 使用建议
1. 每份讲义先看“知识图谱”，再进入模块学习和实操。
2. 每次实操保留操作记录、问题记录、修复记录。
3. 每份讲义完成后做一次自测与复盘。
